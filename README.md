# Calculadora do curso
Projeto feito para o curso de análise de dados EBAC, dentro do módulo introdutório

O objetivo aqui é colocar em prática todos os conhecimentos adquiridos na parte introdutória do curso de análise de dados.
Foi feita uma calculadora onde são feitas diversas operações de forma simultânea depois de receber dois números inteiros por parte do usuário
Logo em seguida, é questionado ao usuário se ele gostaria de repetir a solicitação (por meio de um loop) ou se deseja encerrar ali mesmo.
